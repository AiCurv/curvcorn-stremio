# Curvcorn Stremio

Stremio addons collection by Curvcorn.

## Addons

### HDThot Addon (Curvcorn)
Browse models, categories, popular and latest videos from HDThot.

**Features:**
- Home: Latest videos
- Popular: Filter by Day, Week, Month, All Time
- Categories: 65 categories (OnlyFans, ManyVids, ASMR, etc.)
- Models: Browse and search models

**Install:** `https://curvcorn-hdthot.vercel.app/manifest.json`

### W1MP Addon
Original W1MP Stremio addon.

**Install:** `https://curvcorn-w1mp.vercel.app/manifest.json`

### Dailymotion Addon
Watch videos from Dailymotion directly in Stremio — browse by category, trending, featured, and navigate to creator channels.

**Features:**
- Featured & Trending catalogs
- 17 category channels (News, Sports, Music, Tech, Gaming, etc.)
- Creator channel deep-linking (click an owner to see their latest 20 uploads)
- Clean HLS Auto streaming

**Install:** `https://dailymotion-stremio-addon.vercel.app/manifest.json`
